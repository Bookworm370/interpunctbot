Fun


@inter·punct has a variety of fun commands.

Configuration


Fun commands are enabled by default.

fun


Usage: ip!fun enable|disable
Permissions: runner needs manage_bot.
enables or disables fun and games
you
ip!fun disable

inter·punct BOT
Fun is no longer allowed on this server.

editmsg


Usage: ip!editmsg message link
editmsg [link to a message from @inter·punct].
Right click / Tap and hold the message from @inter·punct that you want to edit, and select 'Copy Message Link' to get the message link.

viewmsgsource


Usage: ip!viewmsgsource message link
viewmsgsource [link to a message]. it will give you a link to the source markdown of the message.
Right click / Tap and hold the message you want to view source to, and select 'Copy Message Link' to get the message link.
