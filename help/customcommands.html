Custom Commands


inter·punct has the ability to create custom commands and quote lists.

command add


Usage: ip!command add commandname text...
Permissions: runner needs manage_bot.
add a custom command

command remove


Usage: ip!command remove command name
Permissions: runner needs manage_bot.
Remove a command
you
ip!command remove motivation

inter·punct BOT
Command removed

lists list


Usage: ip!lists list
list all quote lists on the server
you
ip!list lists

inter·punct BOT
@you, Lists:
motivation: https://pastebin.com/NFuKYjUN

Quote Lists


ip!lists list — list all quote lists on the server
ip!lists add listname https://pastebin.com/url — Add custom command list.
ip!lists edit listname https://pastebin.com/url — Edit custom command list.
ip!command remove command name — Remove a command
